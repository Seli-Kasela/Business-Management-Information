# Installation

Short instructions for setting up a development environment. This file is a starting point — update with real commands once the code is implemented.

Requirements
- Docker & Docker Compose
- Node.js (LTS) and npm or yarn
- Python 3.10+

Local development (placeholder)
```bash
# clone
git clone https://github.com/Seli-Kasela/Business-Management-Information.git
cd Business-Management-Information

# start services (after implementation)
docker compose up --build
```

Environment
- Copy `.env.example` to `.env` and set DATABASE_URL and JWT_SECRET
